# Did I Lock It
Did I Lock It is a simple application for Android which helps you remember if you locked your door by letting you set the locked/unlocked state in the app.

## Building
1. Clone the repository using Git or download the ZIP using the website
2. Open the project in Android Studio
3. Navigate to the Build menu on the toolbar
4. Press "Build APK"
